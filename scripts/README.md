# Scripts

Planned script groups:

- `collection/`: ingest official current and historical exhibitor sources.
- `enrichment/`: normalize company identity and public business contacts.
- `validation/`: schema, evidence, deduplication, and status checks.
- `reporting/`: progress, coverage, opportunity, and quality reports.

Scripts must be reproducible, preserve source provenance, avoid restricted-platform bulk scraping, and never perform outreach.
