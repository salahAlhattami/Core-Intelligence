# Core Intelligence

**Core Intelligence** is the shared operational repository for **Core Intelligence Platform (CIP)** — Core Media's Riyadh exhibition and exhibitor intelligence system.

GitHub is the single source of truth shared by Salah, ChatGPT, and Codex.

## Operating model
- **ChatGPT:** strategy, task definition, research review, quality control, and PR review.
- **Codex:** collection, implementation, enrichment, validation, tests, reports, and PRs.
- **Salah:** priorities, approval of blocked/sensitive decisions, and final merges.
- **GitHub:** durable shared memory, evidence, and audit trail.

## Intelligence model
1. Confirmed current-edition exhibitors.
2. Candidates from at least the last two completed editions, targeting three where available.
3. Public-source business contact enrichment and Core Media opportunity scoring.

Historical participants are never treated as current confirmed exhibitors without current-edition evidence. If the current directory is unavailable, historical collection begins immediately rather than waiting.
